# Little Lemon React Native Capstone Project
## Getting started
### Make sure you have the required software installed on your computer
Before attempting the following steps, make sure that you have installed git, npm and node on your device.
### Download and clone this repository to your local machine
You can use the "git clone" command for this.
### Install all the required dependencies
Navigate to the project's folder in a Terminal (for Linux and MacOS) or a Powershell app (for Windows). Or, you may alternatively, open the project folder with VSCode. Once you do that, type the following command into the command line: "npm install" -- this will install all the required dependencies.
### Start the metro server
Once you're done with the previous step, type the following command into the command line: "npm start" and press enter. Then wait until the Metro server starts up. You will then need to open the Expo Go app on your mobile device (this app was tested on an Android phone). If you don't have Expo Go app yet, go to your app store and install it. 
### Use the Expo Go app
Inside the Expo Go app, select the "Scan QR code" option and scan the QR code that was displayed in the command line / terminal. Keep in mind, that you will need to be connected to your home Wi-Fi network in order for this to work. Then, after some time, the app will start.
